## What I'm Building

### [Codebrain](https://github.com/devmkpro/codebrain) — Multi-Agent AI IDE

> The IDE where AI agents work together, not alone.

Electron desktop app where multiple AI agents (Claude, Gemini, MIMO, Codex) orchestrate in a visible terminal grid. Share memory, coordinate squads, automate browsers, and learn patterns — all in real-time.

**Stack:** `Electron` `React 19` `TypeScript` `Tailwind` `Zustand` `SQLite` `MCP`

```
244 MCP tools  ·  Multi-agent squads  ·  Browser automation (CDP)
Shared memory  ·  Pattern learning    ·  LSP integration
Session checkpoints  ·  Goal/Judge system  ·  Mission management
```

| Feature | What it does |
|---------|-------------|
| Multi-Agent Orchestration | Orchestrator + specialized workers (backend, frontend, tester) |
| Shared Memory (SQLite) | FTS5 search, knowledge graph, PageRank ranking |
| Browser Automation | 60 CDP tools — click, fill, screenshot, network intercept |
| Mission System | Group panes by mission, switch context instantly |
| Goal/Judge Model | Independent judge evaluates if autonomous work is truly done |
| Session Compaction | LLM summarizes old messages when context fills up |
| LSP Integration | goToDefinition, findReferences, hover — semantic code navigation |
| Marketplace | Install agents and skills with 1 click from registry |
